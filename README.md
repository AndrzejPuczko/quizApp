### Installation
If you really want to edit this app just:

1. Clone the repo
   ```sh
   git clone https://github.com/AndrzejPuczko/quizApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
